# square_helmholtz_coil
MATLAB simulation of three perpendicular square Helmholtz coil pairs.

Currently works for symmetrical HC

Computed with B(z)=𝜇_0/2 𝑁𝐼𝑅^2 (1/〖(𝑅^2+〖(𝑧+𝑑/2)〗^2)〗^(3/2) +1/〖(𝑅^2+〖(𝑧−𝑑/2)〗^2)〗^(3/2) )
